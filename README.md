RP - Laboratuvar Kaynak Planlama Sistemi (Vize Ödevi)
Selamlar hocam, bu projeyi laboratuvar envanterini daha kolay yönetmek ve öğrencilere bilgisayar zimmetleme işlemlerini takip etmek için geliştirdim. Hem backend tarafında veri yönetimini hem de frontend tarafında kullanıcı dostu bir arayüzü birleştirmeye çalıştım.

🛠 Neler Kullandım?
Backend: ASP.NET Core Web API kullanarak verileri yönettim.

Veritabanı: Pratik olduğu için SQLite kullandım, veriler LRP.db üzerinde tutuluyor.

Frontend: HTML, CSS ve JavaScript (Fetch API) kullanarak sayfalar arası geçişleri (SPA mantığıyla) hazırladım.

🚀 Nasıl Çalıştırılır?
Öncelikle LRP-Backned klasörüne terminalden girip dotnet run komutunu yazarak API'yi ayağa kaldırmanız gerekiyor.

API çalıştıktan sonra index.html üzerinden yönetim paneline, student.html üzerinden de öğrenci sayfasına ulaşabilirsiniz.

📌 Notlarım
Veritabanı bağlantılarını ve tabloları (Laboratuvarlar, Bilgisayarlar, Öğrenciler) sizin istediğiniz formatta hazırlamaya calıstım.
